# Design Context

## Users
South African NSC Grade 12 students preparing for matriculation exams. They're stressed, time-constrained, and need clarity. The interface should feel encouraging and confident, not chaotic or overwhelming.

## Brand Personality
**Trustworthy, calm, capable** — like a skilled tutor who knows exactly what you need.

Three words: **Clear, focused, supportive**

## Aesthetic Direction
- Clean, minimal interface with purposeful use of color
- Typography-driven hierarchy (Geist body, Playfair Display headings)
- Floating glass-morphism navigation (liquid, organic feel)
- Light/dark mode support
- Focus on content clarity over decorative elements

## Design Principles
1. **Content first** — Every element serves the learning goal
2. **Calm productivity** — No visual noise, just clarity
3. **Progressive disclosure** — Simple first, sophisticated on demand
4. **Respect the user** — No condescending copy, no gamification gone wrong
5. **Performance is UX** — Fast interactions build trust

## Technical Design System

### Core Stack
- **Components**: shadcn/ui (built on Radix UI primitives)
- **Styling**: Tailwind CSS 4 with custom Tiimo 3.0 theme
- **Animations**: Framer Motion + CSS keyframes
- **Icons**: Huge Icons, Phosphor, Lucide

### Color System
- **Primary**: Lavender (#9F85FF)
- **Background**: Cream (#FAF8F5) / Dark gray (#1F1F1F)
- **Success**: Green (#5CB587)
- **Text**: Gray dark (#464644) / White (dark mode)
- **Subject Colors** (8 NSC subjects):
  - Math: #F2C445 (yellow)
  - Physics: #48A7DE (blue)
  - Life Sciences: #5CB587 (green)
  - Chemistry: #14B8A6 (teal)
  - Accounting: #F472B6 (pink)
  - English: #818CF8 (indigo)
  - Geography: #2DD4BF (cyan)
  - History: #FB923C (orange)

### Typography
- **Headings**: Playfair Display (serif)
- **Body**: Geist Sans
- **Numeric/Code**: Geist Mono
- **Math/Science**: Noto Sans Math

### Spacing & Sizing
- **Base unit**: 4px
- **Border radius**: 6px → 36px (7 steps: xs, sm, md, lg, xl, 2xl, full)
- **Shadows**: 3 levels (tiimo, tiimo-lg, tiimo-xl)

### Navigation
- Floating liquid glass navbar (always visible)
- Global hamburger menu in AppLayout.tsx
- Main content requires pb-40 padding-bottom for navbar clearance
