# Design Context

## Users
South African NSC Grade 12 students preparing for matriculation exams. They're stressed, time-constrained, and need clarity. The interface should feel encouraging and confident, not chaotic or overwhelming.

## Brand Personality
**Trustworthy, calm, capable** — like a skilled tutor who knows exactly what you need.

Three words: **Clear, focused, supportive**

## Aesthetic Direction
- Clean, minimal interface with purposeful use of color
- Typography-driven hierarchy (Geist body, Playfair Display headings)
- Floating glass-morphism navigation (liquid, organic feel)
- Light/dark mode support
- Focus on content clarity over decorative elements

## Design Principles
1. **Content first** — Every element serves the learning goal
2. **Calm productivity** — No visual noise, just clarity
3. **Progressive disclosure** — Simple first, sophisticated on demand
4. **Respect the user** — No condescending copy, no gamification gone wrong
5. **Performance is UX** — Fast interactions build trust
