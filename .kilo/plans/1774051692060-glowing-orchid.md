# UI/UX Polishing Plan - Lumni AI

## Overview
Comprehensive UI/UX polish for this South African Grade 12 education platform, covering typography consistency, navigation, components, landing page, dashboard, animations, and mobile responsiveness.

## Current State Analysis

### Design System (Tiimo 3.0)
- **Fonts**: Geist (body), Geist Mono (numeric), Playfair Display (headings), Noto Sans Math (equations) - ✅ Properly configured
- **Colors**: Lavender primary (#9F85FF), cream background, subject-specific pastels - ✅ Comprehensive palette
- **Spacing**: 4pt base system with semantic aliases - ✅ Well established
- **Radius**: 6px to 9999px scale - ✅ Consistent
- **Shadows**: Three-tier system - ✅ Implemented
- **Dark mode**: Full support with "energetic gray" variant - ✅ Working

### Key Pages
- Landing page with hero, features, stats, subjects, testimonials
- Dashboard with 15+ widgets (growth map, tasks, subjects, achievements)
- 40+ other screens (quiz, lessons, flashcards, tutoring, etc.)

## Phase 6: Animation Refinement

### 6.1 Motion Patterns
- Use Tiimo design system animations:
  - `animate-tiimo-float`: 4s float animation
  - `animate-tiimo-pulse`: 2s pulse
  - `animate-tiimo-spin`: 20s rotation
  - `animate-tiimo-shimmer`: Loading state

### 6.2 Transition Consistency
- Use `--duration-fast` (150ms), `--duration-normal` (300ms), `--duration-slow` (500ms)
- Use `--ease-tiimo` for bouncy, `--ease-smooth` for smooth

### 6.3 Page Load Animations
- Review framer-motion usage
- Ensure staggered animations don't feel too slow
- Check for layout shift during load
