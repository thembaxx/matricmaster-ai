# Phase 5: Monetization - Design Spec

## Overview
Improve the subscription flow with better paywall, trial flows, and tiered pricing display.

## Goals
1. Create a clear feature gating system
2. Add free trial flow
3. Improve pricing page display

---

## Implementation

### 1. Feature Gating
- Create hook to check subscription status
- Show upgrade prompts for premium features

### 2. Trial Flow
- Free trial for new users
- Trial status indicator

### 3. Pricing Display
- Clear tier comparison
- Current plan indicator
